* [Symfony v5](https://symfony.com/doc/current/index.html)
* [Vue v2](https://vuejs.org/)
* [Bootstrap-vue v2](https://bootstrap-vue.org/)
* Sass
* [Google fonts](https://fonts.google.com/)

## Environnement de production

* Raspberry Pi 4 Modèle B
* Raspbian
* Apache
* Mysql
* Phpmyadmin
